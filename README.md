<h2>1. 안드로이드 앱 개발 준비하기</h2>
- 안드로이드 버전별 특징<br>
- 안드로이드 개발 준비<br>
본격적으로 안드로이드 개발을 공부하기 전에 안드로이드가 사용자 관점에서 어떻게 변화했는지를 가볍게 다뤄본다. 또한 안드로이드 앱을 개발하는 환경도 구축해본다.<br>
	<br>
<h2>2. 첫 번째 앱 만들기</h2>
- 안드로이드 프로젝트 만들기<br>
- 프로젝트 구성<br>
- 그레이들 빌드 시스템<br>
- 앱 실행<br>
간단한 첫 번째 앱을 만들면서 안드로이드 앱 개발의 흐름을 익힌다. 앱을 개발하면서 필요한 기본 개념과 안드로이드 스튜디오의 사용법도 함께 익혀본다. 직접 예제를 만들어보면서 레이아웃을 구성하는 방법과 레이아웃을 자바 클래스에서 불러오는 방법을 배우고, 안드로이드 화면을 구성하는 단위가 무엇이며 화면 간의 이동 방법은 무엇인지 배워보도록 한다.<br>
	<br>
<h2>3. 뷰와 뷰그룹</h2>
- 뷰와 뷰그룹<br>
- 레이아웃 기초<br>
- 간단한 사용자 인터페이스 구성<br>
- 인텐트: 다른 액티비티 실행<br>
안드로이드에서 화면을 구성하는 기본 요소인 뷰(View)와 뷰 크룹(ViewGroup)의 개념을 학습한다.<br>
	<br>
<h2>4. 자주 사용하는 뷰, 유용한 뷰</h2>

- 기타 유용한 뷰<br>
자주 사용하는 다섯 가지 뷰와 기타 유용한 뷰의 기본적인 사용 예를 살펴본다.<br>

<h2>5. 레이아웃: 화면에 뷰를 수놓는 방법</h2>
- LinearLayout (선형 레이아웃)<br>
- RelativeLayout (상대 레이아웃)<br>
- FrameLayout (프레임 레이아웃)<br>
- ConstraintLayout (제약 레이아웃)<br>
	<br>
안드로이드의 레이아웃을 구성하기 위해 필요한 개념인 뷰그룹(ViewGroup)에 대해 배우도록 한다. 대표적인 뷰그룹으로 LinearLayout, RelativeLayout, FrameLayout과 ConstraintLayout이 있다.<br>
